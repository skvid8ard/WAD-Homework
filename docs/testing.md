# Тестирование проекта

Проект покрыт автоматизированными асинхронными тестами с использованием `pytest`.

## Архитектура тестов
- **Фреймворк:** `pytest` + `pytest-asyncio`
- **HTTP-клиент:** `httpx.AsyncClient`
- **База данных:** Временная SQLite в оперативной памяти (`aiosqlite:///:memory:`).
- **Подход:** Подмена зависимостей (`app.dependency_overrides`) для изоляции тестовой среды от продакшен базы данных.

## Запуск тестов

Для запуска всех тестов убедитесь, что виртуальное окружение активно, и выполните:
```bash
pytest -v
```
