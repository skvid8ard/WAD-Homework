## Frontend Framework (SPA)
- **React 18**: Библиотека для построения пользовательских интерфейсов на основе компонентного подхода.
- **Vite**: Сверхбыстрый инструмент сборки и сервер разработки нового поколения.
- **TypeScript**: Строгая типизация для предотвращения runtime-ошибок и описания API-контрактов.
- **Tailwind CSS v4**: Utility-first CSS фреймворк для быстрой стилизации и поддержки Dark Mode.
- **React Router v6**: Библиотека для клиентской маршрутизации (SPA).
- **Lucide React**: Набор современных минималистичных SVG-иконок.

## Backend Framework
- **FastAPI**: Высокопроизводительный асинхронный веб-фреймворк, автоматически генерирующий OpenAPI документацию.
- **SQLAlchemy 2.0**: Современная ORM со строгой типизацией (`Mapped`, `mapped_column`) и поддержкой асинхронности (`asyncpg`).
- **Alembic**: Инструмент контроля версий схемы базы данных (миграции).
- **Redis**: In-memory хранилище. Используется для хранения долгоживущих refresh-токенов (срок жизни 30 дней) и кэширования.
- **Llama-cpp-python**: C++ биндинги для локального запуска и инференса LLM (Large Language Models) на CPU.

## Инфраструктура
- **Docker & Docker Compose**: Контейнеризация сервисов (PostgreSQL, Redis) для унификации среды разработки и продакшена.
- **PostgreSQL**: Основная реляционная база данных (хранение юзеров, чатов, истории сообщений).
- **SQLite (In-Memory)**: Используется изолированно для мгновенного выполнения автотестов (pytest).
