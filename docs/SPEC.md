## Цели спринта:
Переход от локальной разработки к полноценному production-окружению. Упаковка в контейнеры, обеспечение безопасности трафика и настройка реальных сервисов связи.

## Задачи:
1. **Контейнеризация (Docker)**: 
   - Написание Multi-stage Dockerfile для фронтенда (Nginx + React).
   - Dockerfile для бэкенда с поддержкой C++ компиляции (llama-cpp).
   - Настройка Docker Compose для связи Backend, Frontend, PostgreSQL и Redis.
2. **Reverse Proxy (Nginx)**:
   - Настройка единой точки входа на порту 80/443.
   - Проксирование запросов `/api/` на бэкенд и `/` на фронтенд.
3. **Security (HTTPS)**:
   - Интеграция с Certbot (Let's Encrypt) для получения SSL-сертификатов.
4. **Email Service**:
   - Замена `email_service` (заглушки) на реальную отправку через SMTP (SendGrid, Mailgun или личный домен).
5. **CI/CD Foundation**:
   - Подготовка скриптов для автоматического деплоя одной командой.
