# Sprint 8: Авторизация и Анимации (Текущий)
Интеграция framer-motion для плавных переходов страниц.
Настройка Zustand для глобального управления состоянием пользователя (залогинен/нет).
Создание базового API-клиента (shared/api.ts) с автоматической подстановкой JWT токена.
Верстка и логика страниц /login и /register (включая кнопки входа через Google/GitHub).

# Sprint 9: Сердце приложения (Chat UI и Стриминг)
Верстка ленты сообщений (Message Bubbles) и поля ввода с авто-расширением высоты.
Интеграция рендера Markdown (ведь LLM отдает текст с жирным шрифтом и блоками кода).
Написание логики работы с Server-Sent Events (SSE) для чтения потокового ответа из POST /chats/{id}/messages.
Эффект печатной машинки и плавный авто-скролл вниз.

# Sprint 10: Stateful UX (История и Управление)
Оживление Сайдбара: загрузка списка чатов с бэкенда (GET /chats).
Кнопка "Новый чат" и переключение между комнатами.
Загрузка истории старого чата при клике (GET /chats/{id}/messages).
Перехватчики (Interceptors) для автоматического обновления (Refresh) токена, если access_token протух.

# Sprint 11: Полишинг и UX (Предрелиз)
Уведомления об ошибках (Toast notifications), например, при неверном пароле или обрыве сети.
Скелетоны (Skeleton Loaders) во время загрузки истории чатов.
Адаптивность (Mobile Friendly) — чтобы сайдбар прятался под "гамбургер" на телефонах.

# Sprint 12: Production Деплой (Финальная стадия)
Написание многоэтапного (Multi-stage) Dockerfile для сборки React-приложения через Nginx.
Объединение всего в единый docker-compose.yml (Бэкенд + БД + Redis + Фронтенд).
Инструкции по запуску проекта одной кнопкой.
