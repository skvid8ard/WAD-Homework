# Sprint 1: Базовая инфраструктура и настройка

**Статус:** Завершен

## Что сделано:
Заложен фундамент проекта. Определена первоначальная архитектура:
* `core/config.py` — управление переменными окружения через pydantic-settings.
* `core/database.py` — подключение к PostgreSQL (асинхронный движок).
* `core/redis.py` — подключение к Redis.
* `main.py` — минимальная точка входа FastAPI.

Настроен минимальный `docker-compose.yml` для поднятия инфраструктуры:
* `redis:7.4-alpine` с пробросом пароля и сохранением данных на диск.
* `postgres:15-alpine` с персистентным хранилищем.
