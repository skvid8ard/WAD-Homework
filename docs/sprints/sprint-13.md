# Sprint 13: Production Деплой (Финальная стадия)
**Статус:** В процессе (Выполнено 50%) 🚀 | **Версия:** v0.13.0

## Цели спринта:
Переход от локальной разработки к полноценному production-окружению. Упаковка в контейнеры, обеспечение безопасности трафика и настройка реальных сервисов связи.

## Задачи:
- [x] **Контейнеризация (Docker)**: Написание Dockerfile (React + FastAPI/llama-cpp), настройка Docker Compose.
- [x] **Изоляция и Безопасность**: Скрытие БД внутри виртуальной сети, настройка `pydantic-settings` (Fail Fast), единый `.env`.
- [x] **Reverse Proxy (Nginx)**: Проксирование запросов `/api/` на бэкенд и `/` на фронтенд. Восстановление работы Swagger.
- [ ] **Email Service**: Замена `email_service` (заглушки) на реальную отправку через SMTP (aiosmtplib).
- [ ] **Security (HTTPS)**: Интеграция с Certbot (Let's Encrypt) для получения SSL-сертификатов на VPS.
- [ ] **OAuth Configuration**: Настройка боевых `redirect_uri` для Google и GitHub.
- [ ] **CI/CD Foundation**: Скрипты для автоматического деплоя.
